# 📘 Step Motor İleri-Geri 360° Döndürme — Konu Anlatımı

> **Kaynak Dosya:** [MOTOR.pbp](file:///c:/Users/Aleyna/Desktop/denetleyici/MOTOR.pbp)
> **Konu:** Tek fazlı step motor, FOR-NEXT döngüsü, 360° dönüş hesabı

---

## 📌 1. Bu Kod Ne Yapıyor?

1. Step motoru **48 adımda 360°** ileri döndürür
2. Sonra **48 adımda 360°** geri döndürür
3. Bu işlemi sonsuz döngüde tekrarlar

---

## 📌 2. Step Motor Adım Hesabı

```basic
for j = 0 to 47    ' 48 adım = 360°
    gosub ileri
next j
```

Bu motorun **adım açısı = 7.5°** dir.

```
360° ÷ 7.5° = 48 adım
```

| Adım Açısı | Tam Tur İçin Gerekli Adım |
|:---:|:---:|
| 7.5° | 48 |
| 5.625° | 64 |
| 1.8° | 200 |

> [!IMPORTANT]
> **Sınavda sorulabilir:** "Adım açısı 7.5° olan bir step motor kaç adımda tam tur döner?" → **48 adım** (360 / 7.5 = 48)

---

## 📌 3. FOR-NEXT Döngüsü

```basic
for j = 0 to 47
    gosub ileri
next j
```

| Komut | Açıklama |
|:---|:---|
| `FOR değişken = başlangıç TO bitiş` | Döngü başlat |
| `NEXT değişken` | Bir sonraki değere geç |
| `STEP değer` | Artış miktarı (belirtilmezse 1) |

Örnekler:
```basic
for i = 0 to 9          ' 0,1,2,...,9 → 10 kez döner
for i = 10 to 0 step -1 ' 10,9,8,...,0 → geri sayım
for i = 0 to 100 step 5 ' 0,5,10,...,100 → 5'er artar
```

---

## 📌 4. Tek Fazlı Sürme Yöntemi

Bu kodda motor **tek fazlı** olarak sürülür. Tek fazlı = aynı anda sadece **bir bobin aktif**.

### İleri Yön (Bit Sola Kaydırma)

```basic
ileri:
    pulsout portA.2, 100
    i = i << 1             ' Sola kaydır
    if i = 16 then i = 1   ' %00010000 olunca %00000001'e dön
    portB = i : pause 300
return
```

```
Adım 1:  %00000001  → RB0 aktif
Adım 2:  %00000010  → RB1 aktif
Adım 3:  %00000100  → RB2 aktif
Adım 4:  %00001000  → RB3 aktif
Adım 5:  %00000001  → tekrar RB0 (döngü)
```

### Geri Yön (Bit Sağa Kaydırma)

```basic
geri:
    pulsout portA.2, 100
    i = i >> 1             ' Sağa kaydır
    if i = 0 then i = 8    ' 0 olunca %00001000'e dön
    portB = i : pause 300
return
```

```
Adım 1:  %00001000  → RB3 aktif
Adım 2:  %00000100  → RB2 aktif
Adım 3:  %00000010  → RB1 aktif
Adım 4:  %00000001  → RB0 aktif
Adım 5:  %00001000  → tekrar RB3 (döngü)
```

---

## 📌 5. PAUSE Süresinin Etkisi

```basic
pause 300    ' 300 ms bekleme
```

PAUSE süresi motorun **hızını** belirler:
- `PAUSE 100` → Hızlı döner
- `PAUSE 300` → Orta hızda döner
- `PAUSE 500` → Yavaş döner

> [!WARNING]
> PAUSE çok kısa olursa motor adımları kaçırabilir (tork yetmez). Çok uzun olursa gereksiz yavaş döner.

## 📌 6. Sınav İçin Dikkat Noktaları

| Konu | Hatırla |
|:---|:---|
| **Adım açısı** | 7.5° → 48 adım = 360° |
| **FOR-NEXT** | Döngü: `for i=0 to 47 ... next i` |
| **Tek fazlı** | Aynı anda sadece 1 bobin aktif |
| **Sola kaydır** | İleri yön: `i << 1` |
| **Sağa kaydır** | Geri yön: `i >> 1` |
| **Taşma kontrol** | i=16 → i=1 (ileri), i=0 → i=8 (geri) |
| **PAUSE** | Hız kontrolü, ms cinsinden |
